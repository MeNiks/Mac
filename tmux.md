tmux new -s server
Ctrl + b, then % - Horizontal
Ctrl + b, then " - Vertical
Ctrl + b, then arrow keys - Move between panes
Ctrl + b, then w - check window number
Ctrl + b, then window number (0,1,2…) to switch to that pane
Ctrl + b, then x - Kill pane
Ctrl + b, then & - Kill window
Ctrl + b, then d - Detach
tmux attach -t server - Attach to a session
tmux ls - List sessions
